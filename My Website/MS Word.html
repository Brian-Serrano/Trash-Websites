MS Word

Microsoft Word is the document editor that you can take with you on the go.

Writing and collaborating is no longer confined to the office. Edit docs, share notes, craft a powerful cover letter, and collaborate on projects using your mobile device. Documents are easy to create, read, edit and share. Regardless of the format, Microsoft Word is one of the best free writing apps for writing on the go.

Access files and make edits with ease using the Word app. Read documents and open attachments with built-in docs viewer or edit a PDF using Word’s file conversion. Share files and notes in the format you’d like. Create documents on the go with Word’s expert editing tools.

Resume templates, newsletters, brochures and more are accessible in the template gallery. Select the documents you would like and get started.

Blogger, writer, journalist, columnist or project manager: Microsoft Word moves with you.

Write a letter, create templates, read Word documents, edit docs and collaborate with your team anywhere with the docs app that can do it all.

Create Documents

1. Use Microsoft Word as a cover letter creator by leveraging our beautiful templates

2. Documents are easily created for any writing task.

3. Editing with rich formatting & layout.

4. Document editor keeps formats & layouts looking great.

5. Docs app with templates for resumes, cover letters, forms & more.

Read, Write and Edit

1. Read docs, letters, PDFs & scripts on your device in Reading View.

2. Edit documents from PDFs by converting to & from Word documents.

3. PDF file conversion: save as PDF after editing & share the PDF easily.

Collaborate and Share With Anyone, Anywhere

1. Collaborate - comment in your document right next to the text using Office Suite.

2. View any document using the built-in docs viewer.

3. Read & edit documents as a team and stay on top of changes to the text, layout & formatting.

4. Editor history: revert to view earlier drafts & edit docs.

Microsoft Word for Anything

1. Create documents like letters, blogs, scripts, notes, resumes, & more with beautifully designed modern templates.

2. Share documents as a link, email or attachment.

3. Write a letter easily using Word’s templates.

4. Proofread, check spelling & review any document.

Sharing Simplified

1. Share files & collaborate with a few taps.

2. File & document permission management: see who is working on what.

3. Copy files into the body of an email message with its format intact or attach PDFs & docs to the email.

Download Microsoft Word, a writing app free of restrictions free of hassle allowing you to edit documents anywhere.
